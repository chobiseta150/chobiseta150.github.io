JAVA SCRIPT

Concepto

Java Script es un lenguaje de programación encargado en dotar de mayor interactividad y dinamismo a las páginas web. Con este lenguaje de programación se puede crear todo tipo de efectos y animaciones tambien es usado para interaciones con el usuario como apretar un boton o modificaciones en el DOM. Java script es aveces confundido con Java en si pero lo que talvez no sabian es que tienen funciones completamente differentes como acabo de mencionar Java es utilizado para crear paginas web mientras que Java script es utilizado para Videojuegos casi siempre.

Antecedentes

Java script fue introdisido en 1996 com una manera de añadir programas a unas paginas web en el navegador Netscape Navigator. En los primeros dias de WorldWide Web,HTML era bastante simple de utilizar y facil de aprender casi todo lo que necesitaba saber para agrupar paginas web.

Sistemas operativos

Muchas de las funciones de Javascript es crear Sistemas Operativos. Pero que es un sistema operativo? Un sistema operativo es un conjunto de programas que permite manejar la memoria, disco medios de almacenamiento de información y los diferentes periféricos o recursos de nuestra computadora como son el teclado, el mouse, la impresora, la placa de red, entre otros
